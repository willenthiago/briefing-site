# Briefing — Desenvolvimento de Site

Formulário de briefing para coleta de informações antes do desenvolvimento de sites.

## Sobre
Formulário multi-etapas com 10 seções cobrindo negócio, público, visual,
funcionalidades, SEO, infraestrutura, prazo e estratégia.
Gera um resumo copiável ao final.

## Como usar
Envie o link para o cliente preencher antes da reunião de alinhamento.
